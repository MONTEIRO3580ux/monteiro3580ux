🌍 Calculadora de Escala Geográfica

Validações físicas aplicadas.

  • SI (Sistema Internacional): BIPM (Bureau International des Poids et Mesures), padrões ISO 80000.
  • Dados: Padrão IEC 80000-13 para prefixos binários (KiB, MiB) versus decimais SI (kB, MB).
  • Estados Unidos: NIST Handbook 44 para conversões imperiais (lb, oz, ft).
  • Iluminação: Categoria simplificada para Iluminância (Lux, fc) pois fluxo e intensidade dependem de geometria.
  • Temperatura: Zero absoluto validado a 0 K (-273.15 °C).
  • Constantes Físicas:
    • Velocidade da Luz (c) = 299.792.458 m/s (exata).
    • gravidade padrão (gn) = 9.80665 m/s².

📜 Histórico de Versões

v4.0 Arquitetura Modular
  • Código Refatorado: Lógica do aplicativo dividida em módulos separados para melhor manutenção e velocidade.
  • Acessibilidade: Adicionados rótulos (aria-labels) para leitores de tela em botões visuais.
v3.0 Conectividade PWA
  • Aplicativo Instalável: O conversor agora é um Progressive Web App (PWA). Você pode instalá-mo no seu celular ou computador!
  • Suporte Offline: O layout, conversões locais e escalas geográficas funcionam perfeitamente sem internet.
v2.5 Mercado Global
  • Moedas em Tempo Real: Integração com a ExchangeRate-API para buscar taxas de câmbio de todo o mundo.
  • Cache Inteligente: As taxas de câmbio são salvas no navegador por 24 horas para garantir que você possa converter offline.
v2.4 Experiência Expandida
  • Novas Categorias: Culinária (Xícaras, Colheres) e Consumo de Combustível (km/L, MPG).
  • Busca Inteligente: Filtros de texto adicionados aos menus suspensos de unidades.
  • Conversões Rápidas: Painel sugerindo conversões automáticas para unidades comuns da mesma categoria.
v2.3 Atualização Ágil
  • Melhoria de UX: Seleção automática do texto ao clicar no campo de valor.
  • Botão de Troca: Novo design minimalista para o botão de inverter unidades.
  • Copiar Resultado: Novo botão para copiar a conversão para a área de transferência rapidamente.
  • Histórico de Versões: Adicionada esta seção para acompanhar a evolução do projeto.
v2.2 Navegação Aprimorada
  • Validação Física: Bloqueio de temperaturas abaixo de 0K e distâncias negativas.
  • Novas Categorias: Adição de Torque e Radioatividade (Dose vs Atividade).
  • Correção SI vs IEC: Diferenciação correta entre unidades de base decimal (KB, MB) e binária (KiB, MiB) para dados.
v2.0 Expansão Geográfica
  • Novo Recurso: Calculadora de Escala Geográfica adicionada.
  • Notação Científica: Opção para forçar notação científica no resultado.
  • Melhorias: Responsividade aprimorada para dispositivos móveis.
v1.0 Lançamento Base
  • Conversor Universal: Primeira versão com suporte a Múltiplas Categorias (Comprimento, Área, Massa, etc).
  • Histórico: Salvamento de histórico local (LocalStorage).
  • Dark Mode: Suporte a tema escuro/claro.
